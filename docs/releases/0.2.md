# 0.2 Release

## 概览

`0.2` 版本重点处理了自动注册流程中的稳定性问题，并统一了扩展名称与版本显示。

## 本次更新

- 修复 iCloud 邮箱流程在循环结束后残留标签页未被回收的问题，避免邮箱标签持续堆积。
- 修复多轮成功运行后，步骤 1 可能复用旧 ChatGPT / OpenAI 会话，导致注册入口异常的问题。
- 调整步骤 1 的官网打开判定逻辑，只有在官网注册入口真正就绪后才标记完成，减少“页面实际还能打开但流程先报错”的误判。
- 修复步骤 1 打开官网时，Chrome 主 frame 短暂进入错误页就直接触发 `Frame with ID 0 is showing error page` 硬失败的问题。
- 统一扩展名称、清单版本和面板显示版本为 `0.2`。
- 更新版本解析逻辑，侧边栏更新检查现在支持纯数字版本号显示与比较。

## 适合升级的场景

- 自动运行到十几轮后，首页容易出现状态污染或误判失败。
- 使用 iCloud 邮箱注册时，浏览器标签页会越跑越多。
- 希望 Chrome 扩展管理页、扩展面板和版本展示统一为纯数字版本。
