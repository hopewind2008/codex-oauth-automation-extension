# 0.2a Release

## 概览

`0.2a` 是基于 `0.2` 的稳定性修复版本，重点处理步骤 1 打开官网时的等待竞态与日志误导问题。

## 本次更新

- 修复步骤 1 打开 `chatgpt.com` 后，未先等待标签页 `complete` 就进入内容脚本检测，导致页面仍在加载时容易卡入异常等待的问题。
- 修复 `chatgpt.com` 首页在注册流程里先上报为 `chatgpt` source、后续又被流程当成 `signup-page` source 等待，造成“已就绪但仍提示加载中”的假警报。
- 调整步骤 1 的官网入口等待顺序：先等页面完成首轮加载，再检查 `signup-page` 内容脚本和注册入口状态。
- 补充 `chatgpt.com` 首页 source 识别与步骤 1 开页等待顺序的回归测试。

## 适合升级的场景

- 步骤 1 日志里反复出现“ChatGPT 官网仍在加载，正在重试连接内容脚本...”。
- 日志里先出现“内容脚本已就绪：chatgpt”，随后又继续等待 `signup-page`。
- 官网实际能打开，但步骤 1 仍然容易误判或卡住。

## 验证

- 本地全量测试通过：`npm test`
